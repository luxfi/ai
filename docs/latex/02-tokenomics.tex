\subsection{Bitcoin-Aligned Halving Schedule}

Mining rewards (100M per chain) follow Bitcoin's halving:

\begin{equation}
R_{\text{block}}(h) = R_0 \times 2^{-\lfloor h / 210000 \rfloor}
\end{equation}

where:
\begin{itemize}
    \item $R_{\text{block}}(h)$ = reward per block at height $h$
    \item $R_0$ = initial block reward
    \item $\lfloor \cdot \rfloor$ = floor function
\end{itemize}

\subsubsection{Initial Block Reward}

To distribute 100M tokens per chain:

\begin{equation}
\sum_{i=0}^{\infty} 210000 \times R_0 \times 2^{-i} = 420000 \times R_0 = 100,000,000
\end{equation}

Solving for $R_0$:
\begin{equation}
R_0 = \frac{100,000,000}{420,000} \approx 238 \text{ AI per block}
\end{equation}

\subsection{Chain Competition}

Each chain independently sets:

\begin{enumerate}
    \item \textbf{Mining Rates}: Block reward adjustments
    \item \textbf{Compute Prices}: AI inference/training costs
    \item \textbf{GPU Multipliers}: Hardware tier bonuses
    \item \textbf{Treasury Rate}: 2\% default, adjustable
\end{enumerate}

This creates a competitive market where chains optimize for different use cases:
\begin{itemize}
    \item Ethereum: High security, DeFi integration
    \item Base/Arbitrum: Low fees, fast finality
    \item Lux/Hanzo: Native AI optimizations, Warp messaging
    \item Zoo: Research focus, higher treasury allocation
\end{itemize}

\subsection{Treasury Allocation}

Each chain allocates 2\% of mining rewards to treasury:

\begin{equation}
\text{Treasury}(R) = 0.02 \times R
\end{equation}

Per-chain treasury funds:
\begin{itemize}
    \item Lux/Hanzo: Protocol development
    \item Zoo: Open AI research (DeSci)
    \item External chains: Bridge security, audits
\end{itemize}

\subsection{Token Utility}

AI tokens serve multiple functions:

\begin{enumerate}
    \item \textbf{Payment}: Users pay for AI inference/training
    \item \textbf{Governance}: Token holders vote on parameters
    \item \textbf{Staking}: Miners stake for priority/rewards
    \item \textbf{Cross-Chain}: Teleport enables DeFi
    \item \textbf{Research}: Treasury funds open AI
\end{enumerate}

\subsection{Atomic Units}

Like Bitcoin's satoshis, AI tokens have \textbf{neurons}:

\begin{equation}
1 \text{ AI} = 10^{18} \text{ neurons}
\end{equation}

Micropayment examples:
\begin{itemize}
    \item 1 token generation $\approx$ 1,000 neurons
    \item 1 image generation $\approx$ 100,000 neurons
    \item 1 minute speech $\approx$ 10,000 neurons
\end{itemize}
